# QClaw 版本更新日志

## v2.0.1 (2026/03/16) — 性能优化 + 稳定性提升

### ⚡ 性能优化

| 模块 | 优化项 | 说明 |
|------|--------|------|
| **Main 进程** | 日志 5MB 自动轮转 | 防止 qclaw.log 无限增长撑爆磁盘 |
| **Main 进程** | PROXY 日志跳过磁盘写入 | 减少高频 API 代理日志的 I/O 开销 |
| **Main 进程** | 禁用拼写检查 | 对话应用无需拼写检查，节省内存/CPU |
| **前端渲染** | WorkspacePanel 懒加载 | React.lazy() 按需加载，加快首屏启动 |
| **前端渲染** | Notifications 组件 memo | 避免通知区触发不必要重渲染 |
| **后端服务** | 进化引擎延迟 10s 启动 | 避免与服务冷启动争抢 CPU |
| **构建配置** | target 对齐 node20/chrome120 | 启用更多 JS 原生优化，减少 polyfill |
| **构建配置** | 生产关闭 sourcemap | 减少构建产物体积 |

### 🛡️ 稳定性提升

- 新增 `uncaughtException` / `unhandledRejection` 全局保护，服务器遇到未捕获异常不再崩溃
- `gracefulShutdown` 加入 3s 超时保护，SIGTERM/SIGINT 不再卡死
- `before-quit` 进程清理优化：用 `taskkill /T /F` 递归清理子进程树

### ✅ 集成状态确认

全面验证所有功能已正确集成到 QClaw 主界面（Electron 内嵌，无独立浏览器页面）：

|----------|------|------|
| AI 对话 | 主界面 | ✅ |
| 任务列表 | 左侧导航 | ✅ |
| 控制中心总览 | 左侧 ▦ | ✅ |
| 🧬 进化中心 | 控制中心 Tab | ✅ |
| 技能 / MCP / 自动化 | 左侧导航 | ✅ |
| 用量统计 / Agents / 记忆 | 左侧导航 | ✅ |
| 自我进化引擎 API | 后端 12 个端点 | ✅ |

## v2.0.0 (2026/03/16) — 自我进化引擎 + WorkBuddy 风格全面升级

### 🧬 核心新功能：Self-Evolution Engine（自我进化引擎）

QClaw 作为 AI 集合体，现已具备自我生命力——能够定期审视自身代码、发现缺陷、
借助 Ollama AI 生成改进方案，并在安全机制保护下自主更新自己。

#### 自我进化能力

| 能力 | 说明 |
|------|------|
| **自我扫描** | 每 4h 自动扫描全部 JS 文件（7 条规则，检测大函数/空 catch/内存泄漏等）|
| **AI 驱动建议** | 调用本地 Ollama 生成代码改进方案，降级时使用规则引擎兜底 |
| **安全热替换** | 备份 → `node --check` 语法验证 → 写入 → 验证 → 失败自动回滚 |
| **版本自动递增** | 每次进化成功后自动 bump patch 版本，记录进化历史 |
| **定期进化周期** | 每 24h 执行完整进化：扫描 → AI 建议 → 自动补丁 → 备份清理 |
| **安全边界** | 仅修改 server/ 目录，不触及 Electron 主进程，不修改自身 |

#### 进化中心仪表盘（`/dashboard` → 🧬 进化中心）

- 实时引擎状态（运行中/已停止、版本、扫描次数）
- 扫描问题列表 + AI 改进建议双栏对比
- 进化历史版本表格 + 进化日志流
- 自动补丁开关（默认关闭，手动启用）

#### REST API（12 个端点）
`GET/POST /api/evolution/status|scan|evolve|log|history|advice|backups|patch|rollback|config|toggle`

## v1.2.0 (2026/03/16) — OpenClaw-bot-review 深度集成

### 🎯 主要更新

集成 [OpenClaw-bot-review](https://github.com/xmanrui/OpenClaw-bot-review) 全部核心功能，
升级为完整的 Web 仪表盘控制中心（http://localhost:3001/dashboard）。

### ✨ 新增功能

#### 后端 API（`resources/server/routes/dashboard.js`）

| 端点 | 功能 |
|------|------|
| `GET /api/dashboard/sessions` | 会话列表（Gateway实时+本地扫描，支持类型筛选） |
| `POST /api/dashboard/sessions/:id/test` | 单会话连通性测试 |
| `GET /api/dashboard/skills` | 技能列表（内置/扩展/自定义，支持搜索） |
| `GET/POST/PUT/DELETE /api/dashboard/alerts` | 告警规则 CRUD |
| `POST /api/dashboard/alerts/:id/test` | 告警触发测试 |
| `GET /api/dashboard/gateway` | Gateway 实时健康检测 |
| `POST /api/dashboard/platform-test` | 飞书/Discord/Telegram 连通测试 |
| `GET /api/dashboard/stats` | Token+响应时间趋势数据（day/week/month） |
| `GET /api/dashboard/pixel-office` | 像素办公室 Agent 状态 |

#### 前端仪表盘（`resources/server/public/dashboard.html`）

- 🏠 **总览** — Stat 卡片 + Token 趋势迷你图 + 最新摘要 + 实时告警
- 🤖 **机器人** — Agent 卡片墙（emoji、模型、平台绑定、会话数、状态）
- 🧠 **模型** — Provider 和模型列表，支持切换激活 + 健康检测
- 💬 **会话** — 会话列表（类型识别）+ 连通测试 + 批量平台测试
- 📊 **统计** — Token 消耗趋势 + 响应时间趋势 SVG 图表
- 🔧 **技能** — 技能总览 + 搜索筛选 + 分类统计
- 🔔 **告警** — 规则管理 + 触发历史（持久化到 `~/.qclaw/alert-rules.json`）
- 🕹️ **像素办公室** — Agent 像素动画漫步场景
- ⚡ **Gateway 指示器** — 实时 10s 轮询，侧边栏显示在线状态
- 🌓 **主题切换** — 深色/浅色主题
- ⏱ **自动刷新** — 手动/10s/30s/1min/5min

### 🔧 优化

- `dashboard.js` Agent 信息增强：补充 emoji、platforms、sessionCount 字段
- `dashboard.js` 统计增强：新增 avgResponseMs（平均响应时间）
- `index.js` 路由注册修复：`/api/dashboard` 注册位置移至 404 中间件之前
- `index.js` 新增 `/dashboard` 静态文件服务
